# 室内烟雾检测系统简介 python724

## 项目概述

室内烟雾检测系统是基于Python语言的深度学习技术实现的，旨在快速准确地检测室内烟雾情况，确保环境安全。本系统提供全新的源码和数据库，便于用户快速部署和使用。

## 技术栈

本项目采用以下技术栈构建：

- **后端**: Python、Django
- **前端**: Vue.js
- **数据库**: MySQL
- **架构**: B/S架构

## 功能模块

系统主要包括以下功能模块：

- 用户注册与登录
- 烟雾检测
- 数据展示与分析
- 系统管理

## 系统角色

系统分为以下两个角色：

- **管理员**: 负责系统管理、用户管理和数据监控等操作。
- **普通用户**: 可进行烟雾检测，查看检测数据。

## 运行环境

- **后端**: Python 3.x、Django 2.x
- **前端**: Vue.js 2.x
- **数据库**: MySQL 5.x

## 部署步骤

1. 准备Python运行环境
2. 安装依赖库
3. 配置数据库
4. 运行后端服务
5. 部署前端代码
6. 启动Web服务
